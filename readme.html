Chat Application

Your assignment today is to create a Chat client. The goal is, that by the end of this workshop we'll all be talking to each other in the same chatroom.

Links

  • MDN javascript documentation
  • Backbone.js documentation
  • Backbone.js example application
  • Underscore.js documentation

Requirements

  • Firebug (if you're running Firefox)
  • Git + a Github account
  • A local webserver

Order of work

You are completely free to do this in any order you prefer, but here's a suggestion:

  1. Basic stylesheet to make things look ok.
  2. Create bootstrap code to initialize both collections and the connection class. Hardcode the nickname and email address for now.
  3. Also make sure the 'welcome' section is hidden for now
  4. Create a view for the "messages" section.
  5. Create a view for the "inputArea" section.
  6. Create a view for the "userList" section
  7. Create a view for the "welcome" section
  8. Create the application controller/view that's responsible for setting everything up.
  9. If there's time left, make it cooler :)

Tips

  • If you want to connect to the server, you must load the client through a local webserver. It will not work if you run it through the file:// protocol, due to cross-site restrictions.
  • You're using Git, so commit as often as you can!
  • Have fun.. I suppose?
